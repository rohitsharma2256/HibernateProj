# Hibernate ORM Example Project

This project demonstrates how to use **Hibernate ORM (Object Relational Mapping)** in a Java application to perform database operations.

It is a simple Maven-based project that shows how to configure Hibernate, connect to a relational database, and perform persistence operations using entity classes.
